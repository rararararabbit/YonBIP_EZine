二、业务输出预览面板规范

2.1预览面板 / Preview Panel

配置多智能体模式时，在父智能体对话界面提供子智能体调度关系列表，支持@切换子智能“业务输出预览面板”核心能力是将业务分析、执行的结果快速转化为结构化的、可直接预览的完整 HTML 页面。它承担的是“数据 → 可视化语义结构”的最后一环：接收已整理好的分析内容（标题、指标、图表、明细、建议等），并按指定域规范输出纯语义结构的 HTML，不介入数据查询、不生成视觉样式。yon-design-agentic 是本规范的核心实现 Skill，负责在 Skill 调用链中承接上述规范的渲染职责。

2.2 体验侧的分层

“业务输出预览区”规范的体验侧由 yon-design-agentic 以金字塔式三层分层架构实现，自底向上逐层抽象： Tinper agentic 层（底层）：设计语言原子化层，提供本规范所需的基础 CSS（色彩、圆角等）、数据可视化规范、间距/字体/动效等原子级样式定义，以及纯 CSS 实现的组件。支持租户级主题调整。 Design agentic ui skill 层（中层）：提供 HTML 结构模板与关键词匹配规则，按业务类型（审批、表单、列表等）沉淀 UI 技能，形成规范的模板复用体系。支持租户级模板调整。 业务 skill 层（顶层）：面向具体业务场景，激活页面构建规则，实现“业务输出预览区”的最终页面输出。该层直接响应本规范的输出要求。支持具体业务调整。 设计原则：自底向上提供能力（设计语言 → 模板 → 业务规则），自顶向下驱动调整（业务需求 → 模板适配 → 主题定制）。

2.3 结构规范

“业务输出预览区”输出的 HTML 页面遵循固定的一级/二级结构规范，确保所有分析报告具有统一的区块布局。

三、右侧预览区 ECharts 图表能力说明

3.1、概述

右侧预览区集成了 ECharts 全类型图表渲染引擎，基于 yon-design-agentic 公共模板构建，覆盖 5 大类共 13 种图表类型。所有图表颜色通过 CSS 变量动态读取，支持全局主题切换。依赖 yonyoucloud CDN 提供的 yondesign.css、analytics.css 和 echarts.min.js。

3.2、图表类型总览

分类图表类型适用场景
趋势类折线图趋势波动、时间序列分析
对比类柱状图分类数值对比
对比类条形图分类名较长的横向对比
对比类堆叠柱状图构成比例 + 总量对比
占比类饼图占比构成
占比类环形图占比 + 中心数字展示
占比类南丁格尔玫瑰图占比 + 权重双重映射
多维度类雷达图多维度综合评估对比
多维度类散点图二维分布/相关性分析
多维度类漏斗图转化率分析
多维度类热力图二维密度矩阵
状态类仪表盘目标达成度/健康状态
组合图折柱混合图绝对值 + 增长率同图展示

3.3、交互能力

3.3.1 悬停提示 (Tooltip)

鼠标悬停在图表元素上时弹出数据提示框，支持三种触发模式：

触发模式适用图表表现
坐标轴触发 (axis)折线图、柱状图、条形图、堆叠柱状图、折柱混合图悬停坐标轴任意位置，弹出该轴上所有系列的数据
数据项触发 (item)饼图、环形图、玫瑰图、雷达图、漏斗图、散点图悬停单个元素，显示名称 + 数值（+ 百分比）
定点触发 (position: top)热力图提示框固定出现在顶部，显示行列维度 + 数值

各图表的提示内容格式：

  • 饼图/环形图/玫瑰图：名称 + 数值 + 百分比

  • 雷达图/漏斗图：名称 + 数值

  • 散点图：X 值 + Y 值 + 气泡大小

  • 热力图：星期 + 时段 + 数值

  • 仪表盘：名称 + 值%

3.3.2 辅助线 / 轴指示器 (axisPointer)

鼠标悬停时沿坐标轴出现的视觉引导线，精准对齐数据：

类型效果适用图表
十字准星线（默认）X 轴和 Y 轴各一条引导线，交叉定位数据点折线图、柱状图、堆叠柱状图、折柱混合图
阴影条 (shadow)整行灰色半透明阴影，突出当前分类条形图

3.3.3 图例交互筛选 (Legend)

图例支持点击筛选，作为轻量级数据过滤器：

  • 点击切换：点击图例中的系列名称，可即时切换对应数据系列在图表中的显示与隐藏，其余系列自动重绘适配

  • 位置布局：饼图/环形图/玫瑰图中图例纵向排列于左侧，方便长文本展示

3.3.4 高亮强调 (Emphasis)

鼠标悬停时的视觉反馈效果：

效果类型表现适用图表
阴影浮起黑色半透明阴影，产生浮起立体感饼图
静止高亮扇区不缩放放大，仅色块高亮，保持环形稳定环形图
标签放大内部标签字号由 12px 增大到 20px漏斗图
阴影光晕单元格出现阴影光晕，突出当前位置热力图

3.3.5 视觉映射筛选 (VisualMap)

热力图专属的可拖拽数据范围筛选器：

  • 颜色条：图表底部显示 0~100 的渐变色条，直观映射数值大小

  • 拖拽筛选：色条两端手柄可拖拽调整数值范围，图表实时联动，仅显示所选范围内的数据

  • 色阶：6 阶渐变色（浅蓝 → 青 → 绿 → 黄 → 橙 → 红），低温冷色、高温暖色

3.4、展示能力

3.4.1 响应式自适应

浏览器窗口大小变化时，所有图表自动重算尺寸并重绘，保证在任何视口下完整显示。

3.4.2 折线图

  • 平滑曲线：支持 smooth: true 圆滑拐角

  • 面积渐变填充：自上而下线性渐变（25% → 2% 透明度），增强趋势感知

  • 虚线样式：支持多系列中部分使用虚线区分

3.4.3 柱状图 / 条形图

  • 纵向渐变填充（蓝 → 青），增加立体感

  • 顶部圆角（折柱混合图）或右侧圆角（条形图）

  • 横向布局：分类名较长时自动切换条形图

3.4.4 饼图 / 环形图 / 玫瑰图

  • 环形图中心留空（内径 40%，外径 60%），中心可放置总计数值

  • 南丁格尔玫瑰图：扇形半径按数值缩放，占比与权重双重映射

  • 扇区圆角：边缘带 4px 圆角

  • 标签防重叠：环形图通过关闭标签 + 图例替代避免挤压

3.4.5 雷达图

  • 最多 6 个指标维度，支持多系列叠加对比

  • 填充面积对比：当前数据用半透明填充，对比数据用虚线描边

  • 交替背景色：双色交替填充雷达网格

3.4.6 仪表盘

  • 圆环进度条样式

  • 三色分段：040% 红色（预警）、4070% 黄色（注意）、70~100% 绿色（正常）

  • 数值动画：数值变化时的过渡动画

  • 经典指针样式

3.4.7 散点图

  • 气泡大小映射：第三维数据映射为散点半径

  • 多色散点：各点从 8 色调色板随机取色，透明度 0.6

3.4.8 漏斗图

  • 转化漏斗：从曝光到支付逐级收窄

  • 层级渐变：最小宽度不低于 20%

  • 白色内部标签：直观展示各阶段数值

3.4.9 热力图

  • X 轴 12 时段（00:00~22:00，每 2 小时）、Y 轴 7 天

  • 网格分割线清晰区分行列

3.4.10 折柱混合图

  • 双 Y 轴：左轴展示绝对值，右轴展示百分比

  • 柱线叠加：柱状图 + 折线图共享 X 轴，各自独立刻度

  • 圆形数据点：折线节点为 8px 圆形

四、业务输出预览面板无法覆盖的场景说明

以下内容在所有调研产品中一致不被右侧预览区支持，属于行业公认的预览区能力上限：

顺序禁止内容类型原因
1复杂 BI 报表/动态数据大屏渲染引擎复杂度高，预览区资源不足以承载实时数据刷新
23D 模型/工业仿真/大型拓扑图WebGL 资源占用，需要全屏独占式渲染
3工作流配置/系统管理安全隔离要求，低频率操作不应占用预览区
4超过 3 层关联数据下钻认知负荷，列数过多导致信息密度不可控
5未适配规范的自定义组件/第三方插件渲染兼容性，预览区为受限容器
6同时 2 个以上独立抽屉层级管理冲突，多抽屉的焦点和交互逻辑无法保障
